Prompts operativos

Prompt 1

Analiza los archivos cargados y genera un informe técnico preliminar ENOS conforme a los lineamientos oficiales. Identifica amenaza, exposición, vulnerabilidad, capacidades, brechas, acciones prioritarias, responsables sugeridos, medios de verificación y decisiones requeridas. Marca el resultado como borrador técnico sujeto a revisión.

Prompt 2

Genera el aporte institucional para el Plan de Acción ENOS considerando el tipo de institución, sus competencias, los ejes del plan, los datos cargados y las brechas identificadas. Organiza el resultado por: conocimiento del riesgo, monitoreo/alerta, prevención/mitigación, preparación/respuesta y recuperación/seguimiento.

Prompt 3

Construye una matriz de brechas con columnas: territorio, institución, eje del plan, amenaza, elemento expuesto, capacidad existente, brecha, prioridad, responsable sugerido, acción recomendada, plazo, medio de verificación y decisión requerida.

Prompt 4

Convierte esta información en un resumen ejecutivo para autoridad. Responde: qué ocurre, dónde ocurre, qué está expuesto, cuál es la brecha, qué decisión se requiere, qué institución debe intervenir, cuál es la acción prioritaria y qué evidencia respalda la recomendación.

Prompt 5

Revisa este Plan de Acción ENOS frente al formato referencial oficial y los lineamientos estratégicos. Identifica cumplimiento, vacíos, inconsistencias, acciones faltantes, falta de evidencia, brechas no atendidas y recomendaciones de ajuste.

Para automatización real se debe usar API, base consolidada y plantilla Word. Este GPT personalizado es útil para trabajo manual/asistido, no monitorea carpetas ni formularios automáticamente.
