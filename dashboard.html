Brechas críticas y reportes ejecutivos

Este dashboard queda preparado como módulo gerencial. Para datos reales se debe conectar Kobo → base consolidada → tablero. No se publican respuestas sensibles en la plataforma pública.

Verde

Cantón con formularios clave completos, plan generado/en revisión, acciones preventivas y seguimiento activo.

Amarillo

Información parcial, brechas identificadas, plan incompleto o seguimiento mensual pendiente.

Rojo

Brechas críticas, sin plan, sin seguimiento o información insuficiente para decisión.

Gris

Sin registros o sin participación documentada.
